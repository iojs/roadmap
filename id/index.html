Masa Depan yang Cerah dari io.js

Sebuah Roadmap Berbasis Komunitas

Untuk komunitas, dari komunitas

io.js adalah sebuah pengganti untuk node.js dan kompatibel dengan hampir semua modul dalam npm.

io.js adalah sebuah proyek open source berbasis komunitas. Proyek ini dimulai oleh para kontributor utama node.js, tetapi pada beberapa bulan pertama sudah bisa menarik lebih banyak developer yang aktif dibandingkan dengan keseluruhan developer proyek node.js itu sendiri.

Apa saja yang diharapkan dari io.js?

  1. Stabilitas
  2. Transparansi (Debugging dan Tracing)
  3. Streams yang Lebih Baik
  4. Dukungan Jangka Panjang
  5. Mengutamakan standar yang berkelanjutan
  6. Pelokalan

Pertimbangan di atas dikompilasi dari proyek roadmap. Kami masih mengharapkan kontribusi Anda.

Kebijakan Stabilitas

Kami memiliki ekosistem package terbesar dari semua bahasa, yang membuat io.js sangat stabil.

  • Kami tidak menghapus API JS.
  • Sistem pengaturan versi yang semantik (Semantic Versioning)
    • Setiap penambahan API, akan ada perubahan versi minor.
    • Setiap perubahan di API C++ V8 akan ditangani oleh nan dan akan ada perubahan versi minor. Sedangkan peningkatan versi major akan terjadi jika perubahan API terjadi di luar kemampuan nan.

Dari bagian Kebijakan Stabilitas dari Roadmap. Kami masih mengharapkan kontribusi Anda.

Saluran-saluran

Release Build yang memiliki versi apapun.
Harus stabil dan siap untuk produksi.
Canary Build harian dengan Canary V8 & fitur-fitur baru lainnya.
NG API yang berpusat pada Next Generation ES6/7.


Dari proyek Roadmap. Kami masih mengharapkan kontribusi Anda.

NG (Next Generation)

  • Sebuah platform baru untuk aplikasi-aplikasi JavaScript modern.
  • Akan tetap memiliki kompatibilitas dengan versi sebelumnya (backward compatible) tanpa patokan waktu.
  • API-API baru dapat berpusat pada ES6/7 tanpa merusak modul-modul dan API yang lama.
  • Proses pengembangan API yang terpisah dengan proses penyempurnaan untuk platform terkini.

Diskusi tentang NG dapat ditemukan dalam repo NG.

Debugging & Tracing

Suatu inti akan menumbuhkan suatu ekosistem sarana debugging dan penelusuran (tracing) yang besar dan beraneka ragam daripada hanya mendukung high level abstractions seperti domains.

  • Integrasi dengan trace_event (API penelusuran baru untuk V8 & Chrome).
  • Mendokumentasi dan mempelopori sebuah ekosistem debugging.
  • Menyatukan system tracing endpoint (DTrace, LTTng, etc)

Kegiatan ini dipimpin oleh Tracing Working Group. Kami masih mengharapkan kontribusi Anda.

Streams

  • Streams Working Group telah mengambil alih readable-stream, yang adalah modul resmi io.js stream.
  • Memperbaiki semua masalah-masalah kompatibilitas yang ada.
  • Menyederhanakan pembuatan stream untuk menghindari kesalahan pengguna.
  • Menjelajah interface WHATWG Streams dan mengidentifikasi masalah-masalah kompatibilitas.
  • Meningkatkan performa stream.

Kegiatan ini dipimpin oleh Streams Working Group. Kami masih mengharapkan kontribusi Anda.

Dukungan Jangka Panjang

  • Jalur rilis baru akan diluncurkan dengan dependensi yang stabil dan supported.
  • Rilis-rilis patch dari versi sebelumnya akan tetap dipublikasikan selama orang-orang aktif berkontribusi.
  • io.js akan bertanggung jawab terhadap dependensi versi lama seperti V8 yang tampil di rilis-rilis terdahulu ketika sudah tidak ada lagi support dari kontributornya.

Ringkasan dari Kebijakan Stabilitas. Kami masih mengharapkan kontribusi Anda.

Pelokalan

  • Mengurangi dampak dari pemuatan icu (jejak binari dan memori).
  • Memelihara sebuah komunitas pelokalan yang dinamis.
    • Komunitas bahasa yang swatantra (30+ komunitas telah terdaftar sejauh ini)
    • Usaha evangelisasi untuk meningkatkan keanggotaan (160+ anggota sejauh ini)
    • Mengembangkan sarana untuk menerjemahkan website, dokumentasi API, installer dan slide-slide presentasi.

Anda dapat terlibat dengan bergabung dengan komunitas yang sesuai dengan bahasa Anda :)
